Module Contents

.. py:class:: PathMarker(graph, pos, default_edges, default_nodes)

   A class that represents a path marker for a graph.

   .. attribute:: graph

      The graph object.

      :type: object

   .. attribute:: pos

      A dictionary containing the positions of the nodes in the graph.

      :type: dict

   .. attribute:: default_edges

      A list of default edge colors.

      :type: list

   .. attribute:: default_nodes

      A list of default node colors.

      :type: list

   .. attribute:: marked_nodes

      A set of marked nodes.

      :type: set

   .. attribute:: right_marked_nodes

      A set of right-click marked nodes.

      :type: set

   .. attribute:: marked_edges

      A set of marked edges.

      :type: set

   .. attribute:: edge_colors

      A list of edge colors.

      :type: list

   .. attribute:: node_colors

      A list of node colors.

      :type: list


   .. py:method:: mark_node(node)

      Marks a node.

      :param node: The node to be marked.



   .. py:method:: right_mark_node(node)

      Marks a node with a right-click.

      :param node: The node to be right-click marked.



   .. py:method:: un_right_mark_node(node)

      Unmarks a right-click marked node.

      :param node: The node to be unmarked.



   .. py:method:: unmark_node(node)

      Unmarks a node.

      :param node: The node to be unmarked.



   .. py:method:: toggle_node(node)

      Toggles the marking of a node.

      :param node: The node to be toggled.



   .. py:method:: toggle_right_mark_node(node)

      Toggles the right-click marking of a node.

      :param node: The node to be toggled.



   .. py:method:: mark_edge(edge)

      Marks an edge.

      :param edge: The edge to be marked.



   .. py:method:: unmark_edge(edge)

      Unmarks an edge.

      :param edge: The edge to be unmarked.



   .. py:method:: toggle_edge(edge)

      Toggles the marking of an edge.

      :param edge: The edge to be toggled.



   .. py:method:: reset_colors()

      Resets the marked nodes and edges to their default colors.



   .. py:method:: is_node_marked(node)

      Checks if a node is marked.

      :param node: The node to be checked.

      :returns: True if the node is marked, False otherwise.
      :rtype: bool



   .. py:method:: is_edge_marked(edge)

      Checks if an edge is marked.

      :param edge: The edge to be checked.

      :returns: True if the edge is marked, False otherwise.
      :rtype: bool



   .. py:method:: update_plot(nx, plt)

      Updates the plot with the marked nodes and edges.

      :param nx: The networkx module.
      :param plt: The matplotlib.pyplot module.
